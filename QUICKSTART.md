# 🚀 البدء السريع - دليل التثبيت والتشغيل

تريد تشغيل البوت في أقل من 5 دقائق؟ اتبع هذا الدليل خطوة بخطوة!

## ✅ المتطلبات الأساسية

- **Python 3.8+** (تأكد من تثبيته)
  ```bash
  python --version  # تحقق من الإصدار
  ```

- **3 مفاتيح من الإنترنت:**
  1. رمز بوت التلجرام
  2. Chat ID للتلجرام
  3. مفتاح Finnhub API

## 📝 خطوة 1: احصل على رمز بوت التلجرام

1. افتح التلجرام وابحث عن `@BotFather`
2. ابدأ محادثة معه بإرسال `/start`
3. اكتب `/newbot`
4. اتبع الخطوات واختر اسماً للبوت
5. **نسخ الـ Token** (يبدو مثل: `123456:ABC-DEF...`)

## 📍 خطوة 2: احصل على Chat ID

### الطريقة 1: عبر مجموعة
1. أنشئ مجموعة جديدة على التلجرام
2. أضف البوت للمجموعة
3. أرسل رسالة في المجموعة
4. افتح هذا الرابط في المتصفح:
   ```
   https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates
   ```
   (استبدل `[YOUR_BOT_TOKEN]` برمز البوت الخاص بك)
5. ابحث عن `"chat":{"id":` - الرقم بعده هو Chat ID

### الطريقة 2: عبر قناة
1. أنشئ قناة عامة
2. أضف البوت كمسؤول
3. أرسل رسالة عبر `@BotFather` للبوت
4. استخدم نفس الرابط أعلاه للعثور على Chat ID

## 🔑 خطوة 3: احصل على مفتاح Finnhub

1. اذهب إلى https://finnhub.io
2. سجل حساباً جديداً (مجاني)
3. تحقق من بريدك الإلكتروني
4. ادخل لحسابك
5. انقر على **API Key** في الجانب الأيسر
6. **انسخ مفتاح API الخاص بك**

## 💻 خطوة 4: التثبيت على جهازك

### على Windows:
```bash
# فتح Command Prompt أو PowerShell
# انتقل إلى مجلد البوت
cd C:\path\to\stocks-bot

# تثبيت المتطلبات
pip install -r requirements.txt
```

### على Mac/Linux:
```bash
# فتح Terminal
# انتقل إلى مجلد البوت
cd ~/path/to/stocks-bot

# تثبيت المتطلبات
pip install -r requirements.txt
```

## ⚙️ خطوة 5: إعداد البيانات

1. في مجلد البوت، جد ملف `.env.example`
2. انسخه وأعد تسميته إلى `.env`
3. افتح الملف بمحرر نصوص (Notepad, VS Code, إلخ)
4. املأ البيانات:

```env
# معلومات بوت التلجرام
TELEGRAM_BOT_TOKEN=أدخل_رمز_البوت_هنا
TELEGRAM_CHAT_ID=أدخل_Chat_ID_هنا

# مفتاح Finnhub
FINNHUB_API_KEY=أدخل_مفتاح_Finnhub_هنا

# قاعدة البيانات (اتركها كما هي)
DATABASE_URL=sqlite:///stocks_bot.db
```

**مثال:**
```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdef-GHIjkl_MNOpqr_STUvwxyz
TELEGRAM_CHAT_ID=-987654321
FINNHUB_API_KEY=c12a3b4c5d6e7f8g9h0i
DATABASE_URL=sqlite:///stocks_bot.db
```

## 🧪 اختبر البوت

افتح التلجرام وأرسل أوامر للبوت:

1. **للبداية:**
   ```
   /start
   ```

2. **لمعرفة الأوامر:**
   ```
   /help
   ```

3. **لتحليل سهم (مثال: Apple):**
   ```
   /analyze AAPL
   ```

4. **للتحقق من الحلال:**
   ```
   /islamic_check MSFT
   ```

5. **لإضافة سهم للمراقبة:**
   ```
   /watch GOOGL
   ```

إذا كل شيء يعمل ✅ - مبروك! البوت جاهز!

## 🔧 استكشاف المشاكل

### ❌ "No module named 'telegram'"
```bash
# قم بتثبيت المتطلبات
pip install -r requirements.txt
```

### ❌ "Invalid token"
- تأكد من نسخ رمز البوت **بالكامل** من BotFather
- تحقق من عدم وجود مسافات أو أحرف إضافية

### ❌ "Chat not found"
- تأكد من أن Chat ID صحيح وأن البوت موجود في المجموعة/القناة
- الرقم يجب أن يكون سالباً للمجموعات: `-123456789`

### ❌ "Finnhub API error"
- تحقق من صحة مفتاح API
- تأكد من أنك وصلت للحد الشهري (خطة مجانية = 60 طلب/دقيقة)

### ❌ البوت يتوقف بعد قليل
- تأكد من اتصالك بالإنترنت
- جرب إعادة التشغيل
- تحقق من السجلات في ملف `bot.log`

## 📊 تشغيل البوت 24/7

إذا كنت تريد البوت يعمل طول الوقت:

### على Windows:
استخدم **Task Scheduler** لتشغيل `python main.py` كل يوم

### على Mac/Linux:
استخدم **cron** أو خدمة مثل **systemd**

### الخيار الأفضل:
استضيف البوت على سيرفر سحابي (AWS, Heroku, DigitalOcean)

## 💡 نصائح مهمة

1. **لا تشارك مفتاحك**: احرس مفتاح Finnhub API
2. **راقب الحد الشهري**: الخطة المجانية محدودة
3. **اقرأ البيانات بحذر**: التحليل للمعلومات فقط
4. **استشر مستشار مالي**: لا تتخذ قرارات استثمارية بناءً على البوت وحده
5. **احفظ الأسهم المهمة**: استخدم `/watch` للأسهم التي تهمك
