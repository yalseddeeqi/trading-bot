# ✨ ما الجديد في النسخة 2.1

## 🎉 الميزات الجديدة:

### 📅 **التقارير التلقائية اليومية** ⭐⭐⭐
البوت الآن يرسل تقارير **تلقائياً 3 مرات يومياً** بدون ما تطلب:

```
🌅 9:30 AM  ← تقرير فتح السوق
☀️ 1:00 PM  ← تقرير منتصف اليوم
🌆 3:30 PM  ← تقرير قبل الإغلاق
```

كل تقرير يحتوي على:
- ✅ أفضل الفرص الحالية
- 💹 أسعار وتغيرات السهم
- 🧑‍💼 عدد المحللين الموصين بالشراء
- ⚠️ مستوى المخاطر
- 🕌 فحص إسلامي تلقائي

### 🔄 **بيانات حقيقية من السوق** ⭐⭐⭐
- توصيات المحللين الفعلية (ليس بيانات ثابتة)
- أسعار السهم الحقيقية المحدثة
- فرص مختلفة كل يوم حسب السوق
- تحديث فوري للبيانات

### 🔍 **تحليل أي سهم** ⭐⭐
```
/analyze AAPL      ← تحليل Apple
/analyze NVDA      ← تحليل Nvidia
/analyze أي سهم   ← تحليل أي شركة أمريكية
```

### 🕌 **فحص إسلامي ذكي**
- استثناء أسهم التبغ والمقامرة والكحول
- استثناء أسهم الأسلحة والدفاع
- فلترة تلقائية حسب الشريعة الإسلامية

## 🚀 كيفية الاستخدام:

### للتقارير التلقائية:
```env
# ضع في .env:
TELEGRAM_CHAT_ID=رقم_المجموعة_أو_القناة
```

والبوت سيرسل التقارير **تلقائياً** في الأوقات المحددة:
- 9:30 AM ET (توقيت نيويورك)
- 1:00 PM ET
- 3:30 PM ET

**ملاحظة:** التقارير تُرسل فقط أيام العمل (الاثنين-الجمعة)

### للأوامر اليدوية:
```

/opportunities     ← الفرص الحالية
/analyze MSFT      ← تحليل أي سهم
/stocks           ← قائمة الأسهم
/help             ← المساعدة
```
