# 📅 التقارير التلقائية اليومية

البوت الآن يرسل **تقارير تلقائية 3 مرات يومياً** بدون ما تطلب شيء!

## ⏰ أوقات التقارير:

### 1️⃣ **تقرير فتح السوق** - 9:30 AM (توقيت نيويورك)
```
📈 تقرير فتح السوق
━━━━━━━━━━━━━━━━━━━━━
أفضل الفرص في بداية اليوم
```

### 2️⃣ **تقرير منتصف اليوم** - 1:00 PM (توقيت نيويورك)
```
📊 تقرير منتصف اليوم
━━━━━━━━━━━━━━━━━━━━━
تحديث الفرص في منتصف جلسة التداول
```

### 3️⃣ **تقرير قبل الإغلاق** - 3:30 PM (توقيت نيويورك)
```
⏰ تقرير قبل إغلاق السوق
━━━━━━━━━━━━━━━━━━━━━
آخر فرص قبل نهاية اليوم
```

## 📋 محتوى التقرير:
كل تقرير يحتوي على:
- ✅ أسهم بفرص شراء قوية (أكثر من 50% من المحللين)
- 💹 السعر الحالي والتغير
- 🧑‍💼 عدد المحللين الموصين بالشراء
- ⚠️ مستوى المخاطر
- 🕌 فحص إسلامي للأسهم

## 🔧 الإعدادات:

### ✅ الشرط الأساسي:
يجب أن يكون `TELEGRAM_CHAT_ID` موجود في ملف `.env`:

```env
TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_CHAT_ID=your_chat_id
FINNHUB_API_KEY=your_api_key
```

### 📍 كيفية الحصول على Chat ID:

**الطريقة 1: إذا كنت تستخدم مجموعة:**
1. أضف البوت للمجموعة
2. أرسل رسالة في المجموعة
3. افتح هذا الرابط في المتصفح:
```
https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates
```
4. ابحث عن `"chat":{"id":` - الرقم هو Chat ID

**الطريقة 2: إذا كنت تستخدم قناة:**
1. أضف البوت كمسؤول في القناة
2. أرسل رسالة
3. استخدم نفس الرابط أعلاه

### ⚠️ ملاحظة مهمة:
- التقارير تُرسل **فقط أيام العمل** (الاثنين للجمعة)
- التوقيت بـ **توقيت نيويورك (ET)**
- إذا كان هناك Chat ID ولكن خطأ في الاتصال، سيحاول البوت إعادة المحاولة

## 💡 الأوامر اليدوية:

```
/daily_report     - تقرير فوري الآن
/opportunities   - الفرص الفورية

```

## 🔔 تفعيل/تعطيل التقارير التلقائية:

### ✅ التفعيل:
تأكد من:
1. `TELEGRAM_CHAT_ID` موجود بـ `.env`
2. البوت موجود في المجموعة/القناة
3. البوت له صلاحيات الإرسال

### ❌ التعطيل:
احذف أو اترك `TELEGRAM_CHAT_ID` فارغ:
```env
TELEGRAM_CHAT_ID=
```
