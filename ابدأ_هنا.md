## ⚡ الإعداد السريع (5 دقائق):

### 1️⃣ افتح ملف `.env` وأكمل البيانات:

```env
# معلومات بوت التلجرام
TELEGRAM_BOT_TOKEN=أدخل_رمز_البوت_هنا
TELEGRAM_CHAT_ID=أدخل_Chat_ID_هنا

# مفتاح Finnhub
FINNHUB_API_KEY=أدخل_المفتاح_هنا
```

### 2️⃣ احصل على البيانات المطلوبة:

#### 🤖 **Bot Token:**
- اذهب إلى @BotFather على Telegram
- أرسل `/newbot`
- اتبع التعليمات
- انسخ الرمز

#### 💬 **Chat ID:**
- أنشئ مجموعة أو قناة
- أضف البوت
- افتح الرابط:
```
https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
```
- ابحث عن `"chat":{"id":` - الرقم هو Chat ID

#### 🔑 **Finnhub API Key:**
- اذهب إلى https://finnhub.io
- سجل حساب (مجاني)
- انسخ مفتاح API

### 3️⃣ شغل البوت:
```bash
python telegram_bot.py
```
